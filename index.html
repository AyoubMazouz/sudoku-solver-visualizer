Controls:

  • Use your mouse to select the a cell, then use the built-in number pad or the number pad on your keyboard ("1", "2", "3"..., "9").
  • Use the backspace button or "0" on your keyboard to delete a cell.
  • The drop-down menu above the board alow you to select different difficulty levels.
  • The "Reset" button clear the board for a new game.
  • Solve button will run a backtrack algorithm that will solve the current board.
  • ("x1", "x2", "x3") by selecting one of them you can control the visualization speed default: "x1".
